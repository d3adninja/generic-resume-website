# Generic Resume Website

A modern, high-performance, and fully responsive resume website template built with the latest web technologies. This project is designed to be easily customizable and deployable as a Progressive Web App (PWA).

## üöÄ Tech Stack

This project leverages a cutting-edge stack for optimal performance and developer experience:

-   **Framework**: [Next.js](https://nextjs.org/) (App Router)
-   **Language**: [TypeScript](https://www.typescriptlang.org/)
-   **Styling**: [Panda CSS](https://panda-css.com/) (Zero-runtime, type-safe CSS-in-JS)
-   **Animations**: [Framer Motion](https://www.framer.com/motion/)
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **PWA**: [next-pwa](https://github.com/shadowwalker/next-pwa) for offline capabilities and installability.
